# Project PACS

This project is a web application for managing histology tissue samples. It provides functionalities for users to view, tag, and comment on tissue sample images.
## Stack

- Frontend: The frontend is built using React, TypeScript, and Vite. It uses various libraries such as MUI for UI components, React Router for navigation, Axios for HTTP requests, and i18next for internationalization.
